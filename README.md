

# PokemonApp

It is intended to write clean code on the Pokemon App. Code tied to OOP, POP, Clean Code principles is written. The app features MVVM architecture.

## Third Party Libraries:
- Alamofire
- Kingfisher

## Design Pattern
- MVVM

## API
- https://pokeapi.co/
